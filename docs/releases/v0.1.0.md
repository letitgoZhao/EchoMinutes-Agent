# EchoMinutes Agent v0.1.0

EchoMinutes Agent v0.1.0 is the first Windows public preview of the local-first AI meeting-minutes desktop workflow.

## Highlights

- Import local audio/video files and copy media into the local workspace.
- Transcribe meetings through DashScope ASR behind the backend provider abstraction.
- Generate Qwen-compatible meeting notes from persisted transcript segments.
- Review transcript cards, rename speakers, edit transcript text, regenerate notes, and save edited Markdown.
- Export edited notes to Markdown, PDF, and Word with local export history.
- Configure DashScope API settings and test ASR/LLM readiness inside the desktop Settings page.
- Inspect backend health, workspace paths, FFmpeg availability, provider readiness, and recent workflow logs.
- Launch the packaged Windows app with a bundled FastAPI backend runtime.

## Windows Install

Download and run:

```text
EchoMinutes Agent Setup 0.1.0.exe
```

After installation, open Settings, enter a DashScope API key, run provider readiness checks, then import a local meeting audio or video file.

## Notes

- The app is local-first, but not fully offline: ASR media and transcript text may be sent to the configured providers.
- Speaker diarization quality depends on provider/model behavior and the meeting audio.
- The Windows installer is unsigned in this preview, so Windows may show a trust prompt.
