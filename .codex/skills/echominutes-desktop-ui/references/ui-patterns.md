# Desktop UI Patterns

## Transcript Card

Each segment should show:

```text
[Speaker 1] [00:00:03 - 00:00:12]
Segment text...
```

Expected actions:

- rename speaker
- jump to timestamp when media controls exist
- copy text
- mark important
- show low-confidence hint when available

## Note Editor

The note editor should support:

- Markdown editing
- preview when practical
- save
- regenerate
- export
- insert reference from transcript in later iterations

## Export Modal

Export format:

- Markdown
- PDF
- Word

Export source:

- current edited version
- generated original version

Export location:

- default local exports directory
- user-selected directory when native dialog support exists
